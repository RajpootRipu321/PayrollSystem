# Payroll Management System (Java Swing + JDBC)

A desktop-based Payroll Management System built using **Core Java**, **Swing UI**, and **JDBC**.  
This project performs full **CRUD operations** (Create, Read, Update, Delete) on employee payroll data stored in a relational database.
